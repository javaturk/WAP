Ch02 Web Applications Basics

  • BasicServlet

    Has doGet(), doPost() and doHead() methods and prints out headers. Use WebClient for Post and Head methods.

  • OnlyServletvia GET and via POST

    Implements Servlet interface, overrides service() method and prints out info on the ServetRequest object.


Ch03 Basics of HTTP

  • util.echoserver.EchoServer

    runs as a server and listens to port 8888 and prints whatever it gets. Before clicking the link launch it!

  • WebClient

    is a simple browser that let's you send HTTP requests manually and shows the response received from the server as both text and html.

  • HttpClient

    A class with a main method that connects to a URL on the web and prints out headers and HTML content downloaded using streams.

Ch05 Retrieving Information

  • InitialParameterServlet

    Loads initial parameters from web.xml through ServletConfig.

  • InitialParameterServlet3

    Loads initial parameters using annotations through ServletConfig.

  • ContextParameterServlet

    Loads initial parameters from web.xml through ServletContext.

  • ServletContextServlet

    Lists ServletContext attributes set by the servlet container.

  • AttributeSenderServlet

    Sends an object through ServletContext.

  • AttributeReceiverServlet

    Receives an object through ServletContext.

  • AttributeRemoverServlet

    Removes the attribute "name" added to ServletContext by AttributeSenderServlet.

  • ClientInformationServlet

    Lists all client information on HttpRequest.

  • RequestHeadersServlet

    Lists all HTTP headers on HttpRequest.

  • BrowserCheckerServlet

    Check user's browser.


Ch06 Sending The Response

  • StatusCodesServlet

    Shows how to use status codes.

  • TargetServlet

    Prints its referrer. It is redirected by RedirectingServlet

  • RedirectingServlet

    Redirects to TargetServlet using sendRedirect() method.

  • ContentLengthServlet

    Enter different values for content length and see its effect.

  • FileServerServlet

    Choose among different types to load files. It uses MIME types for content type. Bot setContentType() and setHeader() are used.

  • BufferingServlet

  • AutoFlushingServlet

    Run this servlet on Opera web browser to see the flushing effect.

  • ClientSideCachingServlet

    Output is cached at the browser at first access and then subsequent accesses serve it from the cache. Use Chrome's inspect or Firefox Firebug to see caching.

  • UnavailableServlet

  • FirstServlet

    Includes or forwards to SecondServlet

  • SecondServlet

    Included or forwarded to by FirstServlet

  • IncludedServlet

    Included servlet.

  • ForwardedServlet

    Forwarded servlet.

  • RequestDispatcherInclude

    Includes IncludedServlet.

  • RequestDispatcherForward

    Forwards to ForwardedServlet.

  • RequestAttributeSender

    Forwards to RequestAttributeReceiver and passes attributes using request object.

  • ServletExceptionProducer

    Servlet that is shown when a ServletException occurs. Click here to produce a ServletException

  • HiddenPageServingServlet

    forwards to WEB-INF/hidden.html which can not be accessed directly. Files under WEB-INF directory aren't served directly.


Ch07 Handling HTML Forms

  • GetFormServlet

    An HTML page that submits to GetFormServlet using GET method.

  • PostFormServlet

    An HTML page that submits to PostFormServlet using POST method.

  • ParameterServlet

    Displays a form in doGet() method and then submits to itself using POST method.

  • DefaultValuesFormServlet

    Displays a form in doGet() method with default values and then submits to itself using POST method.

Ch10 Servlet Filters

  • AllFilter, AllFilter3

    is configured for /*.

  • LogFilter

    is configured for /* and logs everything.

  • SelamFilter

    is configured for /SelamServlet.

  • RequestFilter

    is configured for /RequestFilterServlet.

  • RequestFilterServlet

    Prints when doGet() is called.

  • ResponseFilter

    is configured for /ResponseFilterServlet.

  • IncludeFilter

    is configured for /IncludedServlet when included.

  • ForwardFilter

    is configured for /ForwardedServlet when forwarded.


Ch11 Servlet 3.0: Advanced Features

  • SelamServlet3

    SelamServlet configured as a 3.0 servlet through WebServlet annotation.

  • InitialParameterServlet3

    Loads initial parameters using annotations through ServletConfig. Parameters are registered through WebInitParam in WebServlet annotation.

  • DynamicServletRegistration

    is a ServletContextListener and registers following three entitites:

    • DynamicServlet

      is registered dynamically.

    • DynamicFilter

      is registered dynamically. Notice that DynamicFilter will filter DynamicServlet and log to "C:\logDynamic.txt".

    • DynamicRequestListener

      is registered dynamically. Notice that DynamicRequestListener will listen to request events.

  • FileUploadServlet

    FileUploadServlet is a servlet that is configured with @MultipartConfig. To upload a file go upload.jsp

  • PrimeNumberCounterServlet

    Counts all prime numbers up to given limit.

  • AsynchPrimeNumberCounterServlet

    Counts all prime numbers up to given limit using asynchronous servlet.
