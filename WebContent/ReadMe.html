In order for the examples to run correctly please make following changes to the project:

Make sure that "srcDir", "jspDir", and ""web.xml" init parameters of the servlet org.javaturk.wap.util.SourceCodeServlet have the path of "src", "jsp" directories and "web.xml" file in their values, respectively. Only after making these changes would the "Source Code" links in the codes work.

And read Readme.txt in jsp\ch08 to make some db configuration so that leagues.jsp work correctly.

For org.javaturk.wap.ch10.LogFilter make sure that Java has rights to create a file called log.txt in C:.

Consider using a network monitor to understand what happens under the hood. For this purpose you can use tools such as Firefox Developer Edition or Chrome Developer Tools.
